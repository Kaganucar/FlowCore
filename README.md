# FlowCore

FlowCore is a production-oriented ASP.NET Core Web API project built using Clean Architecture principles.

## Architecture

* **FlowCore.Core** → Domain entities and business rules
* **FlowCore.Application** → Use cases, DTOs, service interfaces
* **FlowCore.Infrastructure** → Database access, external services
* **FlowCore.API** → HTTP API layer

## Tech Stack

* ASP.NET Core Web API (.NET 8 LTS)
* Entity Framework Core
* Clean Architecture
* JWT Authentication (planned)
* Role-based Authorization (planned)
* Global Exception Handling (planned)
* Serilog Logging (planned)
* Docker Support (planned)
* Unit Testing (planned)

## Goal

This project demonstrates **production-style backend architecture** and is designed as a portfolio project for backend developer roles.
